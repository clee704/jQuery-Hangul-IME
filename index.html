jQuery 한글 입력기

한/영 변환: Shift+Space, 두벌식/세벌식 변환: Alt+Shift+Space

북마클릿

한글 입력기 버튼을 상단 북마크바에 끌어 넣으세요. 한글을 입력하고 싶은 페이지에서 북마크를 클릭하면 입력기가 활성화됩니다.

사용법

먼저 jQuery를 제외한 jQuery 한글 입력기가 의존하는 라이브러리가 포함된 jquery-hangulime-standalone.min.js 파일을 받아 jQuery보다 나중에 로드되도록 페이지에 삽입합니다. jQuery 한글 입력기는 jQuery 셀렉터가 반환하는 엘리먼트 컬렉션이 사용할 수 있는 hangulime 메서드를 제공합니다. 예를 들어 페이지 내의 모든 <textarea>에서 한글 입력기를 사용하려면 다음과 같이 실행하면 됩니다. 

$('textarea').hangulime();

인자가 없는 것과 인자로 'init'를 넘기는 것은 같은 효과가 있습니다. 다음과 같이 'destroy' 인자를 넘기면 한글 입력기를 사용하지 않습니다. 

$('textarea').hangulime('destroy');
